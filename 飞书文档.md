# 🐾 Codex 像素桌宠生成器 · 使用文档

> 一句话：把**任意一张动物/角色图片**，变成一只**复古像素风桌面小宠物**，
> 直接装进 **Codex / Claude Code**，它会随 AI 的工作状态切换动作。

---

## 一、这是什么

最近 OpenAI Codex 出了「**宠物模式（Pets）**」——写代码时桌面上养一只「赛博桌宠」，
还能实时显示 AI 在干嘛（生成代码时忙碌、出错时难过、可复核时提醒）。

这个工具让你**用自己的图**做一只同款像素桌宠，**不依赖任何 AI、不联网、纯本地**生成。

**效果**：上传一张猫的照片 → 自动产出一只配色一致的像素猫，会呼吸、眨眼、蹦跶、转身。

---

## 二、它怎么工作（原理）

```
你的图片  ──提取主配色──▶  刷到「标准像素宠物模板」  ──套固定动作──▶  72 帧雪碧图
                                                                      （Codex 直接认）
```

- **配色提取**：从图里算出主毛色，自动推导描边色、暗部条纹、浅色肚皮。
- **模板换肤**：把配色刷到一套手绘像素宠物模板上（站立/端坐的正常姿势）。
  - 👉 所以哪怕原图是**躺着/睡着**的，出来也是**正常站姿**。
- **固定动作**：一套通用动作模板，**换任何动物都复用**，无需重新做动画。

## 三、产出格式（Petdex 标准）

| 项 | 规格 |
|----|------|
| 雪碧图 `spritesheet.webp` | 8 行 × 9 列 = **72 帧**，每帧 **192×208 px** |
| 元数据 `pet.json` | `{ id, displayName, description, spritesheetPath }` |
| 安装目录 | `~/.codex/pets/<slug>/` |
| 行 = 状态 | idle / wave / run / failed / review / jump / extra1 / extra2 |

> 这是 [crafter-station/petdex](https://github.com/crafter-station/petdex) 定义的事实标准，
> **Codex / Claude Code / OpenCode / Gemini CLI 通用**。

## 四、怎么用（3 步）

### 1. 准备
- 一台有 **Python 3.8+** 的电脑
- 安装依赖：`pip install pillow`

### 2. 生成
```bash
# 一键脚本
./make_pet.sh 你的图片.png 小灰 kitty

# 或手动
python3 pixel_pet.py --image 你的图.png --name "小灰" --slug kitty --out out
```
跑完在 `out/kitty/` 得到雪碧图、pet.json、预览页，并自动打开浏览器预览。

### 3. 装进 Codex
```bash
cp -r out/kitty ~/.codex/pets/
```
重启 Codex → **Settings → Appearance → Pets** 里选你的宠物，或输入 `/pet`。

## 五、状态 → 动作对照

| 状态 | 动作表现 | Codex 何时触发 |
|------|----------|----------------|
| **idle** 待机 | 轻呼吸 + 眨眼 | 空闲 |
| **wave** 招呼 | 左右摇摆 | 等待你输入 |
| **run** 奔跑 | 快速弹跳 | 正在生成代码 |
| **failed** 出错 | 泛红 + 闭眼难过 | 任务失败 |
| **review** 复核 | 左右张望 | 结果可复核 |
| **jump** 跳跃 | 抛物线起跳 | 进度推进 |
| **extra1** 转身 | 横向旋转 | 备用 |
| **extra2** 爱心 | 开心蹦 + 爱心 💕 | 备用 |

## 六、参数说明

| 参数 | 说明 |
|------|------|
| `--image` | 输入图片（任意格式；纯色/透明背景效果最佳） |
| `--name` | 宠物显示名 |
| `--slug` | 目录名（英文小写） |
| `--desc` | 描述（可选） |
| `--out` | 输出父目录；**省略则直接装到 `~/.codex/pets/`** |
| `--scale` | 像素放大倍数（默认 7） |

## 七、常见问题

**Q：出来的不太像我的猫本体？**
A：本工具是「**模板换肤**」——配色一致，但身形是统一模板。要「就是这只动物本体」的
像素版，需要接 AI 图像生成（本工具不含，可后续扩展）。

**Q：换条狗、换只鸟，身形会变吗？**
A：暂时不变（都是同一套身形，只换颜色）。要支持不同身形，在 `pixel_pet.py` 里加新的
像素模板即可（仿照里面的 `CAT` 字符画）。

**Q：需要联网或 API Key 吗？**
A：**完全不需要**，纯本地、离线、确定性生成。

**Q：背景没抠干净影响配色？**
A：用纯色/透明背景的图最好。复杂背景建议先抠图（可选用 rembg 等工具）。
